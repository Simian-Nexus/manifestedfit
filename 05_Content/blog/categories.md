# Blog Categories

The post taxonomy for manifestedfit.com/blog. The push-draft script and the future content engine assign these by name (creating them if missing), so keep the names exact.

| Category | What goes here | Typical voice |
|---|---|---|
| **Manifestation** | Intention-setting, visualization, "sleep manifestation," abundance/mindset-of-wanting. | Any; Frankie & Nadia shine. |
| **Mindset** | Mental reframes, confidence, nervous-system and psychology-of-change angles. | Dana (practical), Nadia (warm). |
| **Rituals & Routines** | Morning/evening resets, habit-building, daily practices. | Dana, Rowan. |
| **Gentle Movement** | Low-impact fitness, stretching, yoga/tai-chi cues, breathwork-in-motion. | Dana, Rowan. |
| **Sleep & Rest** | Wind-downs, sleep, recovery, "let the day be enough." | Rowan (calm) especially. |
| **Recipes** | Wellness recipes / the AI-avatar recipe Shorts sub-lane (e.g. the detox smoothie idea). | Frankie (fun), Dana (practical). |

## Rules of thumb
- Assign **one primary** category per post; a second is fine if it genuinely fits. Avoid stuffing all six.
- Keep the list small. New categories only when a real content theme recurs — otherwise use tags.
- "The 5-Minute Morning Reset" post → **Rituals & Routines** (+ Manifestation).
- The 7-Day Reset lead-magnet page is a page, not a post, so it stays uncategorized.
