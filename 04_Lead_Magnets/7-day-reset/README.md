# 7-Day Mind-Body Reset

## Purpose

This is the first interactive lead magnet for Manifested Fit.

It should feel like a small daily ritual builder, not a generic PDF. The goal is to help visitors connect a wellness intention with a short movement practice, a nourishment cue, and a reflection prompt.

## User Flow

1. Visitor opts in on the landing page.
2. Thank-you page links to the interactive reset.
3. Visitor chooses their energy, goal, and available time.
4. The tool generates a simple daily plan.
5. The page offers a transparent affiliate CTA for deeper personal growth work.

## Future Improvements

- Save generated plans locally or by email.
- Add a printable PDF export.
- Add a 7-day email companion.
- Add a quiz result segment for better email personalization.
